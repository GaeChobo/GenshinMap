# 내 원신 맵

원신 인터랙티브 맵 (가메닷 맵스 스타일) — 1단계: 정적 버전

## 실행 방법

`index.html`을 더블클릭하면 브라우저에서 바로 열립니다. 서버 설치 불필요.

## 폴더 구조

```
GenshinMap/
├── index.html          ← 실행 파일
├── css/style.css       ← 디자인
├── js/app.js           ← 앱 로직
├── lib/                ← Leaflet 지도 라이브러리 (오프라인 동작)
├── maps/               ← 지도 이미지 (현재는 임시 이미지)
└── data/
    ├── maps.js         ← 지도 목록
    └── markers/        ← 지도별 마커 데이터 (지도마다 파일 1개)
        ├── teyvat.js
        └── enkanomiya.js
```

## 새 지도 추가하는 법 (원신 업데이트 시)

1. 지도 이미지를 `maps/` 폴더에 넣기 (PNG/JPG/SVG 아무거나)
2. `data/maps.js`에 한 줄 추가:
   ```js
   { id: "newarea", name: "새 지역 이름", image: "maps/newarea.png" },
   ```
3. 끝. 마커 파일은 없어도 빈 지도로 열리고, 관리자 모드로 바로 마커를 찍으면 됨.

> Claude Code 세션에서 "OOO 지역 추가해줘"라고 하면 이미지 확보부터 등록까지 대신 해줌.

## 마커 추가하는 법

1. 상단 **[관리자 모드]** 클릭
2. 오른쪽 아래 패널에서 종류 선택
3. 지도를 클릭하면 그 자리에 마커 추가
4. **[마커 내보내기]** → 다운로드된 파일을 `data/markers/`에 덮어쓰기
5. **[커스텀 초기화]**로 임시 데이터 정리

## 성능 설계 (나중에 무거워지지 않도록)

- **지도별 데이터 분리**: 선택한 지도의 마커 파일만 로드. 지도가 100개여도 초기 로딩 동일
- **캔버스 렌더링**: `preferCanvas` 옵션으로 마커 수천 개도 부드럽게 동작
- **대형 맵 타일링**: 티바트 전체맵 같은 초대형 이미지는 타일 분할 예정 (보이는 부분만 로드)
