# Claude 디자인 리뷰 요청 문구 (복붙용)

아래 전체를 Claude(디자인)에 붙여넣고, **첨부 스크린샷 7장**(`바탕화면\원신맵_스크린샷\리디자인_적용후\`)을 함께 올리세요.

---

## (여기부터 복사)

지난번 리디자인(Fontaine Frost 테마)을 실제 앱(바닐라 HTML/CSS/JS + Leaflet)에 **그대로 적용 완료**했어. `css/style.css` 교체 + 테마 토글/모바일 드로어 스니펫 + 진행도 바(`--pct`)/필터 접기(`.grp.collapsed`) 훅까지 다 반영했고, 목업 전용 요소(가짜 지도·데모 마커)는 뺐어. 실제 데이터(맵 7종, 마커 수만 개, 실제 카테고리 아이콘)로 렌더한 결과를 첨부 스크린샷으로 보낼게. **한 번 더 리뷰해서 다듬을 점을 알려줘.**

**첨부 스크린샷**
1. `01_라이트_티바트` — 라이트 테마, 티바트 대륙 오버뷰(워프만 켜진 기본 상태)
2. `02_라이트_필터펼침` — 워프 지점 그룹 펼친 사이드바(하위 카테고리 + 아이콘)
3. `03_라이트_마커팝업` — 확대해서 물방울 핀 마커가 보이는 상태
4. `04_라이트_관리자` — 관리자 모드 패널
5. `05_다크_티바트` — 다크 테마 오버뷰
6. `06_모바일_지도` / `07_모바일_필터드로어` — 모바일(오프캔버스 드로어)

**특히 봐줬으면 하는 점 (디자인 판단이 필요)**
1. **지도 자체가 어두운 톤**이야(게임 원본 맵). 라이트 테마의 밝은 글래스 패널과 대비가 큰데, 이 조합이 괜찮은지 / `--map-shade`(지도 위 워시)를 라이트에서 조절해 조화를 높일지.
2. **마커 밀집 시 물방울 핀이 많이 겹쳐** 보여(오버뷰에서 워프만 켜도 수백 개). 다음 단계로 **클러스터링**(가까운 마커를 묶어 "2/3" 수집/전체 배지로 표시, 원신맵스 방식)을 넣을 예정이야. 이 **클러스터 배지의 디자인**(모양·색·수집 진행 표기)을 제안해줘.
3. **마커 아이콘 대비** — 호요랩 카테고리 아이콘이 흰색/단색 글리프가 많아. 물방울 핀 배경(`--panel-solid`) 위에서, 특히 다크 테마에서 아이콘이 충분히 보이는지. 필요하면 핀 배경/아이콘 처리 제안.
4. **마커 팝업에 "리뷰/댓글" 섹션 추가 예정** — 지금은 "먹었음/삭제"만 있는데, 아래에 유저 리뷰(공략 팁) 목록 + 입력이 들어갈 거야. 팝업 안에 이 리뷰 영역 레이아웃/스타일을 제안해줘.
5. **진행도 바(`#stats`)** — 상단 바 뱃지가 라이트/다크에서 잘 읽히는지, 라벨과 바 색 조화.
6. **물방울 핀 정렬** — 핀의 뾰족한 끝(tip)이 정확한 좌표를 가리키도록 앵커를 tip 기준으로 바꾸는 걸 검토 중(현재는 중앙 앵커). 이건 코드로 처리 가능하지만, 핀 형태/크기와 함께 보면 좋을 것 같아.

**잘 된 점**(유지) — 상단바 엠블럼·드롭다운·진행도 바, 글래스 사이드바 + 그룹 접기, 재생성 자원 그룹의 좌측 액센트, 물방울 핀 + 아이콘, 라이트/다크 토글, 모바일 드로어. 전부 의도대로 동작해.

위 6가지 위주로 다듬은 CSS(및 필요하면 클러스터 배지·팝업 리뷰 영역용 마크업/스타일)를 제안해줘. 기존 id/class 훅은 그대로 유지해줘.
